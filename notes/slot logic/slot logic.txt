Slot parts:
* slot definitions on a template
* bindings to slots via $slot
* filling out slot values via template elements

resolve statically:
when we first hit a template:
* get the slot definitions
* fill in the slots with the template props, any slot without an input prop uses the default value
pass the updated slots to the child as the slot values

when we hit a slot binding (in yui_bind!):
* check if the slot values have an input
  if yes, resolve as a YuiSlotBinding
  otherwise error

if we hit a template when slot values already exist, need to merge them!
